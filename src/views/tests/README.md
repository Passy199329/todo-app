Todo Application

A simple Todo App built with Node.js, Express, MongoDB, and EJS. Users can register, login, and manage tasks in three states: pending, completed, deleted. Each user can only see and manage their own tasks.

Features

User authentication (register & login)

Create, update, delete tasks

Task status management: pending, completed, deleted

Simple EJS-based UI

Error handling and logging

Tech Stack

Node.js | Express | MongoDB | Mongoose | EJS | bcrypt | express-session

Installation

Clone the repo:

git clone https://github.com/YOUR_USERNAME/todo-app.git
cd todo-app


Install dependencies:

npm install


Create .env with:

PORT=3000
MONGO_URI=your_mongodb_connection_string
SESSION_SECRET=supersecret


Run the app:

npm run dev


Visit: http://localhost:3000

Usage

Register → Login → Add / Update / Delete tasks → Logout
